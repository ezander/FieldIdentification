Uncertainty quantification, parameter and field identification methods

By Noemi Friedman and Elmar Zander

This website is a collection of tutorials for uncertainty quantification and parameter/field identification. The website serves for students, researchers and engineers wanting to learn uncertaintiy quantification and Bayesian inversion. The tutorials present how to carry out the analysis throughout toy examples. The tutorial files apply the SGLIB open source MATLAB library package.

This website is also addressed to the readers of our chapter "N. Friedman, E. Zander, C. Zoccarato and H. G. MAtthies: A Worked-out Example of Surrogate Based Bayesian Parameter and Field Identification Methods". The codes for all the examples in the chapter can be found here.

Getting started with SGLIB

In order to be able to run the examples, you will need to download or clone the SGLIB library package, version SGLIB-testing. The package can be downloaded or cloned from the github repository. (Click on the "Code" buttom to download the files or clone by issuing the command on the command line
git clone git://github.com/ezander/sglib-testing
see more about git e.g. here.)

There is a STARTUP.M file in SGLIB. When MATLAB is opened directly from the SGLIB-TESTING directory this file is automatically run, and adds to path the important folders, that you will need for being able to run the tutorials. If you open MATLAB from a different folder, you will have to start by running the STARTUP.M file. If something does not work, you can contact Noemi (e-mail: n.friedman@ilab.sztaki.hu).
