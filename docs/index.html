Uncertainty quantification, parameter and field identification methods

  • By Noemi Friedman and Elmar Zander
  • This website is a collection of tutorials for uncertainty quantification and parameter/field identification. The website serves for students, researchers and engineers wanting to learn uncertaintiy quantification and Bayesian inversion. The tutorials present how to carry out the analysis throughout toy examples. The tutorial files apply the SGLIB open source MATLAB library package. In order to be able to run the examples, you will need to download or clone the SGLIB library package, version SGLIB-testing. The package can be downloaded or cloned from the github repository. (Click on the "Code" buttom and either download the files or clone by git, see more in e.g. )
  • This website is also addressed to the readers of "N. Friedman, E. Zander, C. Zoccarato and H. G. MAtthies: A Worked-out Example of Surrogate Based Bayesian Parameter and Field Identification Methods". The codes for all the examples in the chapter can be found here.
